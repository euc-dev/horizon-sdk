# v1.Disconnect

Closes the underlying channel connection. You can call this method on either the plug-in or the application side.

This function is a member of `VDPService_ChannelInterface`.

## Method Signature
```
Bool (*v1.Disconnect)(void);
```

## Parameters

None

## Return Values

| Value | Description |
| ----- | ----------- |
| TRUE | Call succeeded. |
| FALSE | Call failed. |
