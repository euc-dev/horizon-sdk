# VDPScreenCapture Functions 

With `VDPScreenCapture_Interfac`e functions, you can access the image of the entire remote desktop, an area of the desktop or the windows that belong to a particular application. The vdpScreenCapture.h header file defines the set of functions to use in order to support screen capture functionality in an application.
