# Guest Overlay Sinks

The functions in this section are members of `VDPOverlayGuest_Sink`.

- [`v1.OnOverlayReady`](v1.OnOverlayReady.md)
- [`v1.OnOverlayRejected`](v1.OnOverlayRejected.md)
